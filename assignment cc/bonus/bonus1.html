HTML Tags

This is a quick-reference guide for common HTML tags.

Basic Tags

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Text Formatting

<h1>Heading 1</h1>
<p>This is a paragraph.</p>
<p><strong>Bold Text</strong></p>

Lists

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First Item</li>
  <li>Second Item</li>
</ol>

CSS Properties

This section covers some common CSS properties.

Text Properties

body {
  color: #333;
}

h1 {
  font-size: 24px;
}

Box Model

.container {
  padding: 10px;
}

.box {
  margin: 20px;
}

Background and Borders

.header {
  background-color: #f0f0f0;
}

.image {
  border: 1px solid #ccc;
}

Display and Positioning

.flex-container {
  display: flex;
}

.absolute {
  position: absolute;
  top: 0;
  left: 0;
}
